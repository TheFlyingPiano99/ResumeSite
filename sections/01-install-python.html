Telepítsünk!

Ahhoz, hogy használni tudjuk a Pythont a gépünkön, be kell installálnunk azt. Ehhez látogassunk el a Python hivatalos weboldalára! Navigáljunk a Downloads fülre! Ugyan itt egy nagyon csábító gomb kínálja magát, amivel elindítható a letöltés, azt javaslom, hogy ne engedjünk azonnal a csábításnak. Először nézzük meg a jóval kisebb Python for Windows linket, ami közvetlenül a nagy sárga gomb alatt található. A link egy hosszú listához irányít. Ezek különböző Python verziók letöltési linkjei. Itt fel merül a kérdés: "Minek ennyi verzió? Miért nem elég a legújabbat támogatni?" A válasz erre az, hogy a különböző verziók esetenként nem teljesen kompatibilisek. Egy régebbi verzióra írt program lehet, hogy már nem fut az újabb verzióval. A megfelelő verzió használatával elkerülhetőek a kompatibilitási problémák. Láthatjuk, hogy a verziók három egymásután írt, pontokkal elválasztott számból állnak. Például 3.10.6. Ez egy megszokott konvenció a szoftverek változatainak megkülönböztetésére. Az első szám a MAJOR verzió, a második a MINOR verzió és a harmadik a PATCH szám. Ezek csökkenő nagyságrendű frissítéseket jelölnek. A témáról részletesebben ezen a linken olvashatsz. Python esetében számtalan különböző verzió érhető el, de ezek alapvetően két nagy verzió valamilyen alverziói. A két nagy verzió a Python 2 és Python 3. Ezek között vannak jelentősebb szintaktikai eltérések. A 2.X.X verziók egymással jobbára kompatibilisek. Ez igaz a 3.X.X verziókra is. Máris sok volt a verziókból? Akkor kattintsunk a Latest Python 3 Release - Python 3.10.6 linkre. Ez ugyanaz a verzió, mint amit a nagy sárga gomb kínált a legelején. Most viszont már okosabbak lettünk a verziózás területén. Görgessünk le a files szekcióhoz. Itt a legalsó, Windows installer (64-bit) lehetőséget válasszuk! (Megjegyzés: Amennyiben 32 bites rendszerünk van, akkor az eggyel feljebbi, 32 bites változatot válasszuk! A Windows változatáról a Start menü / Settings / System / About / Device specifications / System type ad pontos információt.)

A megfelelő installálót futtassuk letöltés után! Válasszuk az Install now lehetőséget. Ez az alapértelmezett helyre fog telepíteni, és felrak néhány kiegészítő segédprogramot (PIP, IDLE). Az installáció végén fel fogja kínálni a Disable path length limit lehetőséget. Erre is kattintsunk rá! Adminisztrátori jogosultság megadása után (értsd jóvá kell hagyni egy felugró dialógus ablakban) elvégzi ezt a beállítást is. Ez után bezárhatjuk a telepítőt a close gombra kattintva.

Miután befejeződött a telepítés. Start menüből elérhetőnek kell lennie egy Python 3.10 (64-bit) programnak. Ezt futtatva egy konzol nyílik. Itt máris lehet python utasításokat írni. Próbáljuk meg begépelni a print("Hello World!") sort, majd adjunk Entert. Ez a következő sorba kiírja az üdvözletet. Íme az első Python programunk! Ez a konzol azonban elég fapados. Nem is fogjuk többet használni.

Időzzünk még el egy rövid ideig a frissen installált Pythonunk vizsgálgatásával! Zárjuk be az ablakot, majd keressünk rá ismét a Python 3.10 (64-bit) alkalmazásra a Start menüben! Most ne indítsuk el, hanem jobb klikk és Open file location. Most mindezt mégegyszer: A megnyílt fájlböngészőben jobb klikk a Python 3.10 (64-bit)-en, Open file location. (Azért volt szükség ennek kétszeri elvégzésére, mert a Start menüben csak egy hivatkozás hovatkozása szerepel.) Végre megérkeztünk a Python installáció helyére. Itt már egy python.exe indítót láthatunk. Ez az a futtatható állomány, ami a Start menüben látott hivatkozásra kattintva elindul. Érdekes megfigyelni, hogy ennek nevében már nem szerepel a verzió. Inkább a könyvtár neve Python310. Ebbe a mappába fognak kerülni a további Python programkönyvtárak, amelyeket majd a későbbiekben töltünk le. Hogyha egy kicsit szétnézünk itt, láthatjuk, hogy van egy Doc alkönyvtár, amiben egy fájl található. Ezt megnyitva egy navigálható dokumentációt kapunk. Ez esetleg hasznos lehet. Mondjuk érdemesebb a következő linkhez fordulni segítségért.
